## Why AI agents need L402

AI agents call APIs constantly. Today they can't pay for them — they need a human to set up a credit card, billing account, and subscription. That breaks autonomy.

L402 solves this natively:

1. Agent calls your API
2. Gets a `402 Payment Required` with a Lightning invoice
3. Pays it from its own wallet (e.g. Blink, Phoenix, Alby)
4. Retries with proof of payment (`Authorization: L402 <macaroon>:<preimage>`)
5. Gets the data — **no human, no card, settles in under 1 second**

## Setting up your paid API

<CodeGroup>
```typescript Express (TypeScript)
import express from "express";
import { l402, BlinkProvider } from "l402-kit";

const app = express();
const lightning = new BlinkProvider(
  process.env.BLINK_API_KEY!,
  process.env.BLINK_WALLET_ID!,
);

// AI agents call this — 21 sats per request
app.get("/ai/weather", l402({ priceSats: 21, lightning }), (_req, res) => {
  res.json({ temp: 22, unit: "C", city: "São Paulo" });
});

app.listen(3000);
```

```python FastAPI (Python)
from fastapi import FastAPI, Request
from l402kit import l402_required, BlinkProvider
import os

app = FastAPI()
lightning = BlinkProvider(
    api_key=os.environ["BLINK_API_KEY"],
    wallet_id=os.environ["BLINK_WALLET_ID"],
)

@app.get("/ai/weather")
@l402_required(price_sats=21, lightning=lightning)
async def weather(request: Request):
    return {"temp": 22, "unit": "C", "city": "São Paulo"}
```

## Agent client implementation

This is what the agent-side code looks like — the pattern that any AI framework can implement:

```typescript
import type { LightningWallet } from "./wallet"; // your wallet integration

/**
 * Fetch a paid API endpoint. Automatically pays the invoice if needed.
 * Compatible with any L402-protected URL.
 */
async function fetchL402(url: string, wallet: LightningWallet): Promise<unknown> {
  // 1. First attempt — no auth
  const res = await fetch(url);

  if (res.status === 402) {
    const { invoice, macaroon } = await res.json() as {
      invoice: string;
      macaroon: string;
      price_sats: number;
    };

    // 2. Pay the invoice autonomously
    const preimage = await wallet.payInvoice(invoice);

    // 3. Retry with proof of payment
    const paid = await fetch(url, {
      headers: { Authorization: `L402 ${macaroon}:${preimage}` },
    });

    return paid.json();
  }

  return res.json();
}
```

## LangChain tool example

Wrap any L402 API as a LangChain tool so your agent can call it mid-chain:

```typescript
import { DynamicTool } from "@langchain/core/tools";

const weatherTool = new DynamicTool({
  name: "get_weather",
  description: "Returns current weather. Costs 21 sats per call.",
  func: async (city: string) => {
    const result = await fetchL402(
      `https://your-api.com/ai/weather?city=${city}`,
      wallet, // your LightningWallet instance
    );
    return JSON.stringify(result);
  },
});
```

## OpenAI function calling example

```python
import openai
import json

tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Returns current weather for a city. Costs 21 sats.",
            "parameters": {
                "type": "object",
                "properties": {
                    "city": {"type": "string", "description": "City name"},
                },
                "required": ["city"],
            },
        },
    }
]

def call_paid_tool(name: str, args: dict) -> str:
    if name == "get_weather":
        response = fetch_l402(
            f"https://your-api.com/ai/weather?city={args['city']}",
            wallet,
        )
        return json.dumps(response)

messages = [{"role": "user", "content": "What's the weather in Tokyo?"}]
response = openai.chat.completions.create(
    model="gpt-4o",
    messages=messages,
    tools=tools,
)

# Handle tool call + auto-pay
tool_call = response.choices[0].message.tool_calls[0]
result = call_paid_tool(tool_call.function.name, json.loads(tool_call.function.arguments))
```

## Go and Rust for high-performance agents

For AI agent infrastructure built in Go or Rust, l402-kit provides native SDKs:

```go
// Go — net/http (also works with Chi, Gorilla, Echo)
import l402kit "github.com/shinydapps/l402-kit/go"

mux.Handle("/ai/weather", l402kit.Middleware(l402kit.Options{
    PriceSats:             21,
    OwnerLightningAddress: "you@yourdomain.com",
}, weatherHandler))
```

```rust
// Rust — axum
use l402kit::{l402_middleware, Options};
use std::sync::Arc;

let opts = Arc::new(Options::new(21).with_address("you@yourdomain.com"));
let app = Router::new()
    .route("/ai/weather", get(weather_handler))
    .route_layer(middleware::from_fn_with_state(opts, l402_middleware));
```
