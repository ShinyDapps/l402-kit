## O que é l402-kit?

**l402-kit** é um middleware open-source que implementa o protocolo [L402](https://docs.lightning.engineering/the-lightning-network/l402) — o padrão aberto para pagamentos por chamada de API usando Bitcoin Lightning Network.

Você adiciona 3 linhas de código. Quem chama sua API paga em sats. Você recebe em segundos.

```bash
npm install l402-kit    # TypeScript / Express
pip install l402kit     # Python / FastAPI / Flask
go get github.com/shinydapps/l402-kit/go  # Go / net/http
# Rust: l402kit = { git = "https://github.com/shinydapps/l402-kit" }
```

## Como funciona — fluxo completo

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  Cliente / Agente IA                                            │
│         │                                                       │
│         │ 1. GET /sua-api                                       │
│         ▼                                                       │
│  Sua API (l402-kit)  ──── 2. Cria invoice ───► ShinyDapps API  │
│         │                                       (Blink wallet)  │
│         │ 3. Retorna: 402 + invoice BOLT11                      │
│         ▼                                                       │
│  Cliente paga ──────────────────────────────► Lightning Network │
│         │                    (< 1 segundo)                      │
│         │ 4. Envia preimage como prova criptográfica            │
│         ▼                                                       │
│  Sua API verifica: SHA256(preimage) == hash ✓                   │
│         │                                                       │
│         │ 5. Split automático:                                  │
│         │    → 99.7% ──────────────────────► Seu Lightning Addr │
│         │    → 0.3%  ──────────────────────► ShinyDapps (taxa)  │
│         │                                                       │
│         │ 6. Sua API responde: 200 OK + dados                   │
│         ▼                                                       │
│  Cliente recebe o conteúdo                                      │
└─────────────────────────────────────────────────────────────────┘
```

<Note>
**Transparência total:** ShinyDapps recebe o pagamento e envia **99.7%** para seu Lightning Address automaticamente. A taxa de **0.3%** é o que mantém o projeto vivo. Você pode auditar cada linha em [`src/split.ts`](https://github.com/ShinyDapps/l402-kit/blob/main/src/split.ts) e [`backend/api/split.ts`](https://github.com/ShinyDapps/l402-kit/blob/main/backend/api/split.ts).
</Note>

## Quickstart em 2 minutos

### TypeScript

```typescript
import express from "express";
import { l402 } from "l402-kit";

const app = express();

app.get("/premium", l402({
  priceSats: 100,
  ownerLightningAddress: "você@blink.sv", // recebe 99.7% de cada pagamento
}), (_req, res) => {
  res.json({ data: "Pagamento confirmado." });
});

app.listen(3000);
```

### Python

```python
from fastapi import FastAPI, Request
from l402kit import l402_required

app = FastAPI()

@app.get("/premium")
@l402_required(
    price_sats=100,
    owner_lightning_address="você@blink.sv",
)
async def premium(request: Request):
    return {"data": "Pagamento confirmado."}
```

## Por que não Stripe?

| | Stripe | **l402-kit** |
|---|---|---|
| Taxa mínima | **$0,30** | **< 1 sat (~$0,001)** |
| Liquidação | 2–7 dias | **< 1 segundo** |
| Chargeback | Sim | **Impossível** |
| Requer conta | Sim | **Não** |
| Suporte a IA | Não | **Sim, nativamente** |
| Países bloqueados | ~50 | **0 — global** |
| Código auditável | Não | **Sim, open source** |
| Censurável | Sim | **Não** |

## Segurança

Cada pagamento é verificado matematicamente:

```
SHA256(preimage) == paymentHash
```

- Impossível de forjar sem pagar de verdade
- Cada preimage só funciona uma vez (anti-replay)
- Tokens expiram em 1 hora
- Todo o código é auditável em [github.com/ShinyDapps/l402-kit](https://github.com/ShinyDapps/l402-kit)
