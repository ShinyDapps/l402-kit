## What is l402-kit?

**l402-kit** is an open-source middleware that lets any developer add Bitcoin Lightning micropayments to their API in 3 lines of code.

No credit card. No account. No chargeback. No minimum fee. Just sats.

```
npm install l402-kit
```

## The problem it solves

Today, charging per API call is broken:

- **Stripe**: minimum $0.30 per transaction — impossible for micropayments
- **Subscriptions**: users who use little subsidize users who use a lot
- **AI agents**: can't hold credit cards — can't pay APIs autonomously

**l402-kit** is the third option: pay-per-call via the Lightning Network, settling in under 1 second, globally, with no intermediary.
